CORS (Cross-Origin Resource Sharing)
We got an error because our front-end page is on domain localhost, and from there we're trying to call our back-end API on cloudfunctions.net .
By default, we can only call API endpoints from the same domain.
CORS lets us change that default.
It lets us specify rules about where these calls to our API can originate from.
For now, let's set it so we can call our API from any domain.
